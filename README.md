# ⬡ AethosOS
> Built from scratch. Built for security.

## What is AethosOS?

AethosOS is a Linux security distribution built entirely from scratch using LFS (Linux From Scratch). Not based on Debian. Not based on Arch. Every component compiled from source code — from the kernel up.

## A different approach

Most Linux security distributions are built on top of existing distributions. They are powerful tools, but they inherit architectural decisions made by others.

AethosOS is engineered from the ground up:

- **Pure LFS base** — every component compiled from source
- **Local AI agents** — intelligent assistance with HITL Gate
- **Immutable system** — atomic updates via OSTree
- **GPU offloaded interface** — Wayland and Hyprland
- **Reproducible builds** — every release verifiable

Built for security professionals who want to understand and control every layer of their platform — from the kernel up.

## AI Agents

Intelligent agents assist the operator throughout the security engagement. Local by default. Always.

DISCOVERY → active host identification
RECON     → ports, services and versions
WEB       → web application analysis
EXPLOIT   → locked until explicit double approval
REPORT    → real-time automatic report generation

Cloud mode available as advanced option under operator's explicit responsibility.

## Philosophy

1. Everything compiled from source — no external binaries
2. AI always local by default — cloud is your choice
3. HITL Gate — the human approves every real action
4. Immutable system — signed atomic updates
5. Total transparency — every decision documented

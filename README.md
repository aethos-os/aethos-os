# ⬡ AethosOS
> Built from scratch. Built for security.

## What is AethosOS?
AethosOS is a Linux security distribution 
built entirely from scratch using LFS.
Not based on Debian. Not based on Arch.
Every component compiled from source code.

## Why not Kali, Parrot or BlackArch?

| Feature | Kali | Parrot | AethosOS |
|---|---|---|---|
| Base system | Debian | Debian | LFS — own |
| Local AI agents | No | No | Yes |
| Immutable system | No | No | Yes — OSTree |
| Reproducible build | Partial | Partial | Total |
| GPU offloaded UI | No | No | Yes |

## AI Agents
Local by default. Always.
Cloud mode available as advanced option
under operator's explicit responsibility.

DISCOVERY → active hosts
RECON → ports and services
WEB → web applications
EXPLOIT → locked until double approval
REPORT → real-time automatic report

## Philosophy
1. Everything compiled from source
2. AI local by default — cloud is your choice
3. HITL Gate — human approves every action
4. Immutable system — signed atomic updates
5. Total transparency — every decision documented

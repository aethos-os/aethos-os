Most security distributions are built on top of existing ones. AethosOS is different — every component compiled from source code, nothing inherited.

La mayoría de distribuciones de seguridad se construyen sobre otras existentes. AethosOS es diferente — cada componente compilado desde código fuente.

⚙️
PURE LFS BASE
BASE LFS PURA

Built with Linux From Scratch. Every component compiled from verified source code. No Debian. No Arch. No inherited decisions.

Construido con Linux From Scratch. Cada componente compilado desde código fuente verificado. Sin Debian. Sin Arch.

🤖
LOCAL AI AGENTS
AGENTES IA LOCALES

5 specialized agents assist during security engagements. Local by default. Always. Your data never leaves your machine.

5 agentes especializados asisten durante engagements. Local por defecto. Siempre. Tus datos nunca salen de tu máquina.

🛡️
IMMUTABLE SYSTEM
SISTEMA INMUTABLE

System base is read-only. Updates via OSTree — atomic, signed, rollback-capable. The system never breaks mid-update.

El sistema base es solo lectura. Actualizaciones via OSTree — atómicas, firmadas, con rollback. El sistema nunca se rompe.

👁️
HITL GATE
HITL GATE

Human In The Loop — every real action proposed by AI requires explicit operator approval. The human is always in control.

Humano En El Bucle — cada acción real propuesta por IA requiere aprobación explícita del operador. El humano siempre controla.

🎨
GPU OFFLOADED UI
INTERFAZ GPU

Beautiful interface via Wayland + Hyprland. All rendering on the GPU. CPU and RAM stay free for your security tools.

Interfaz hermosa via Wayland + Hyprland. Todo el renderizado en GPU. CPU y RAM libres para tus herramientas de seguridad.

🔍
REPRODUCIBLE BUILDS
BUILDS REPRODUCIBLES

Every release verifiable. SHA256 + GPG signed. Anyone can verify the complete build chain. Total transparency.

Cada release verificable. SHA256 + GPG firmado. Cualquiera puede verificar la cadena completa. Transparencia total.

Core principles
Principios fundamentales

Philosophy that never compromises

Filosofía que nunca se compromete

01
EVERYTHING FROM SOURCE
TODO DESDE SOURCE

No external binaries. Every component compiled from verified source code with SHA256 checksums.

Sin binarios externos. Cada componente compilado desde código fuente verificado con checksums SHA256.

02
AI LOCAL BY DEFAULT
IA LOCAL POR DEFECTO

LLM runs on Unix socket. No network access. No cloud. Your engagement data never leaves your machine.

LLM corre en Unix socket. Sin acceso a red. Sin nube. Los datos del engagement nunca salen de tu máquina.

03
HUMAN IN THE LOOP
HUMANO EN EL BUCLE

The AI suggests. The operator decides. Always. No autonomous action without explicit human approval.

La IA sugiere. El operador decide. Siempre. Sin acción autónoma sin aprobación humana explícita.

04
IMMUTABLE SYSTEM
SISTEMA INMUTABLE

Atomic signed updates via OSTree. If something fails, automatic rollback. The system never breaks.

Actualizaciones atómicas firmadas via OSTree. Si algo falla, rollback automático. El sistema nunca se rompe.

05
TOTAL TRANSPARENCY
TRANSPARENCIA TOTAL

Every decision documented. Every component auditable. Built in public. Nothing hidden.

Cada decisión documentada. Cada componente auditable. Construido en público. Nada oculto.

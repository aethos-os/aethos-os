# Contribuir a AethosOS

AethosOS acepta contribuciones de todos los niveles.
No necesitas saber LFS para contribuir.

## Formas de contribuir

**Sin conocimiento de LFS:**
- Escribir o mejorar documentación
- Agregar paquetes a aethos-build
- Traducir contenido a otros idiomas
- Reportar bugs o sugerir funcionalidades
- Probar builds y reportar resultados

**Nivel avanzado:**
- Mejorar el sistema base LFS
- Trabajar en los agentes IA (Python)
- Mejoras al hardening del kernel
- Integración con OSTree

## Cómo contribuir

1. Haz fork del repositorio
2. Crea una rama: git checkout -b feature/tu-idea
3. Haz tus cambios
4. Envía un Pull Request
5. Describe qué cambiaste y por qué

## Filosofía

Cada contribución debe respetar la filosofía:
- Sin binarios externos — todo desde código fuente
- IA siempre local por defecto
- Humano siempre en control (HITL Gate)
- Transparencia total

# Contributing to AethosOS

AethosOS welcomes contributions at all levels.
You don't need to know LFS to contribute.

## Ways to contribute

**No LFS knowledge required:**
- Write or improve documentation
- Add tool recipes to the build system
- Translate content to other languages
- Report bugs or suggest features
- Test builds and report results

**Advanced:**
- Improve the LFS base system
- Work on AI agents (Python)
- Kernel hardening improvements
- OSTree integration

## How to contribute

1. Fork the repository
2. Create a branch: git checkout -b feature/your-idea
3. Make your changes
4. Submit a Pull Request
5. Describe what you changed and why

## Philosophy

Every contribution must respect the core philosophy:
- No external binaries — everything from source
- AI always local by default
- Human always in control (HITL Gate)
- Total transparency
